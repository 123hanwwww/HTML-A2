# Inventory Management App
An inventory management application developed based on Angular 17, integrating all functions into the main interface to provide an intuitive inventory management experience. 
## Features
- Inventory Management: Add, edit, and delete inventory items
- Search Function: Search inventory items by name or description
- Statistics Function: View total item count, total quantity, total value, and the most popular items
- Modern Interface: Tabbed layout, responsive design, and attractive visual effects
- Real-time Updates: Automatically update inventory data and statistics after operations

## Technology Stack
- Frontend Framework: Angular 17
- Programming Language: TypeScript
- Styling: CSS
- Data Management: Angular Service (InventoryService)

## Installation and Running
### Prerequisites
- Node.js (v16.0 or higher)
- npm (v7.0 or higher)
- Angular CLI (v17.0 or higher)
